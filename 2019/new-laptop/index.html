New Laptop

发表于2019-06-03|更新于2022-03-11|生活
|字数总计:642|阅读时长:2分钟

　　最近，新购置了一台笔记本电脑，i5 8265u 四核八线程 + 8G DDR4 2666 内存 +MX250 GDDR5 2G独显 + 512G M2 SATA固态；作为一个轻薄本，6h+的续航，1.5kg的重量，稍大的触控板，全金属机身，双风扇双热管，相比于我原先服役了三年的flex-1580，i7 6500u 双核四线程 + 8G DDR3 1600内存 + 940M DDR3 2G独显 + 1T SATA机械硬盘 的配置提升了近两倍，当然flex也有一些其独特的卖点，比如 2in1 的360°翻转屏幕，触控屏，全尺寸键盘等一些细节还是有一定大厂风范；总之，新笔记本还是真香啊~
　　在此基础上，我又添置了 16G DDR4 2666 内存 + PM981 512G NVME 固态，这个最终配置可以说是轻薄本中的王者了。其中PM981算是目前为止体验最惊艳的一个硬件了，3200的读取，1800的写入，简直强到不行，秒开机不是梦，使用体验极致爽滑，强烈推荐笔记本有PCIe 3.0x4 支持nvme插槽的来一块！
　　不过呢，新笔记本也有些小问题，就是cpu睿频导致的电感啸叫，在平衡模式（也就是日常使用）或者开启高性能模式时，电脑会发出高频电流声，就和个电报机一样，这个毛病据说是Intel电源管理的锅，可以通过关闭睿频或者关闭电源管理的C1E来解决这个问题，但是代价就是会损失性能或者续航，这都是我所不愿意看到的，最终，我选择了忽视这个问题，因为其实原先的flex3也有这个问题，只不过情况好一点，平时我都没注意到（也许是当成机械硬盘的声音了），据我所知，这个算是通病，华硕的败家之眼，以及诸如xps13、ThinkPad，甚至MacBook等知名高端机型也都有见网友反应存在这个问题，所以只好妥协了。
　　此外由于更换硬件，我肯定选择将系统装在性能更强的NVME固态上，毕竟原来的SATA协议的固态盘只有520的读取和480的写入，相比于NVME协议的3200读取和1800写入简直就是个弟弟，这又为我带来了一些麻烦，但是好在这机器的BIOS还挺不错的，这麻烦得到了妥善解决，我会在下一篇文章里详细说说这个过程，以及一些经验的分享。
